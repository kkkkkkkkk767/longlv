# 服务流程

## 机票预订流程

### 国内机票

1. **需求确认**：客户提供出行日期、出发地、目的地、人数等信息
2. **查询报价**：根据客户需求查询航班及价格
3. **预订出票**：确认航班后完成预订及出票
4. **售后保障**：提供改签、退票等售后服务

### 国际机票

1. **需求确认**：客户提供行程计划、护照信息等
2. **方案推荐**：根据预算和行程推荐最优航线组合
3. **预订出票**：确认方案后完成预订及出票
4. **行程管理**：协助办理签证、提供行程单等

### 政府采购机票

1. **资质核验**：确认采购单位政府采购资质
2. **合规出票**：按照政府采购规定完成出票
3. **对账结算**：提供规范的结算对账服务

### 团体机票

1. **需求对接**：确认团体人数、行程、预算
2. **团位申请**：向航空公司申请团体优惠价格
3. **统一出票**：完成团体出票及座位安排
4. **变更管理**：统一处理改签、退票等变更

## 用车服务流程

### 接送机服务

1. **预约下单**：客户提供航班号、到达/出发时间、上下车地点
2. **车辆调度**：安排合适车型及司机
3. **接机服务**：司机根据航班动态准时到达接机点
4. **送机服务**：按约定时间上门接送至机场

### 日常用车

1. **预约确认**：客户提供用车时间、地点、车型需求
2. **派单调度**：安排车辆及司机
3. **服务执行**：司机按约提供服务
4. **费用结算**：支持企业月结、单次结算等方式
