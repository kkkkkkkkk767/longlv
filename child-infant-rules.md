# 儿童与婴儿乘机规定

儿童、婴儿乘机的年龄界定、票价标准、服务规定汇总。

> 来源互联网仅供参考，实际以航空公司官网为准。
> 更新日期：2025-12-31

---

## 一、年龄界定

| 类型 | 年龄范围              | 说明                         |
| ---- | --------------------- | ---------------------------- |
| 婴儿 | 出生满14天至未满2周岁 | 部分航司对早产婴儿有额外要求 |
| 儿童 | 年满2周岁至未满12周岁 | —                            |
| 成人 | 年满12周岁及以上      | —                            |

> ⚠️ 出生不足14天的婴儿不予承运。早产婴儿（不足90天）需提供医疗证明。

---

## 二、票价标准

| 类型     | 国内航线          | 国际/地区航线                 |
| -------- | ----------------- | ----------------------------- |
| 婴儿票   | 成人正常票价的10% | 成人正常票价的10%             |
| 儿童票   | 成人正常票价的50% | 成人正常票价的75%或按成人票价 |
| 占座婴儿 | 需购买儿童票      | 需购买占座婴儿票              |

### 注意事项

- 婴儿票不单独占座，由同行成人怀抱乘机
- 婴儿如需单独占座，须购买儿童票
- 儿童购买成人折扣票时，退改签按成人标准执行
- 婴儿免收民航发展基金及燃油附加费
- 儿童享受民航发展基金及燃油附加费减半优惠

## 三、每航班限额

### 婴儿限额

| 机型                | 最大载运婴儿数 |
| ------------------- | -------------- |
| A330/A350/B777/B787 | 15人           |
| A321/C919           | 8人            |
| A319/A320/B737      | 6人            |
| C909                | 4人            |

### 成人携带儿童/婴儿限额

| 组合           | 限额                                         |
| -------------- | -------------------------------------------- |
| 1名成人 + 婴儿 | 最多携带2名婴儿（超过1名时超出的需购儿童票） |
| 1名成人 + 儿童 | 最多携带3名不满5周岁儿童（可含1名婴儿）      |

### 订座要求

儿童与婴儿订座舱位须与同行成人订座物理舱位一致。

## 四、无成人陪伴儿童服务（UM）

### 适用年龄

- 年满5周岁至未满12周岁单独乘机
- 部分航司可接受12-17周岁青少年有偿服务

### 申请要求

| 事项     | 国内航线                     | 国际/地区航线    |
| -------- | ---------------------------- | ---------------- |
| 申请时间 | 航班起飞前48小时             | 航班起飞前96小时 |
| 申请渠道 | 航司直属售票处/客服热线/官网 | —                |
| 提交材料 | 《无成人陪伴儿童乘机申请书》 | —                |
| 数量限制 | 每航班≤12人（5-8岁≤5人）     | —                |

### 接送机要求

- 送机人须年满18周岁，持有效身份证件
- 送机人须在机场停留直至航班起飞
- 接机人须持有效身份证件在到达机场迎接
- 接机人信息需在申请时填写

### 费用

- 国内航线：按成人正常票价50%购票
- 国际航线：按成人适用票价购票
- 无成人陪伴服务本身通常免费（部分航司收费）

## 五、婴儿摇篮服务

| 项目     | 说明                         |
| -------- | ---------------------------- |
| 适用条件 | 婴儿身长≤74cm，体重≤11kg     |
| 申请方式 | 航司官网/APP特殊旅客服务专区 |
| 数量限制 | 每航班数量有限，需提前申请   |
| 适用机型 | 仅限部分宽体机航线           |

---

## 六、儿童安全座椅

- 儿童限制装置须由旅客自行携带
- 尺寸不得超过 43cm × 60cm × 100cm
- 起飞、着陆和地面移动期间不得使用
- 须符合航空安全标准
