# 乘机证件规定

乘坐国内、国际航班所需的证件类型及注意事项。

> 来源互联网仅供参考，实际以航空公司及民航局规定为准。
> 更新日期：2025-12-31

---

## 一、国内航班有效乘机证件

根据《民用航空安全检查规则》，乘坐国内航班的旅客应当出示有效乘机身份证件。

### 中国大陆居民

| 证件类型   | 说明               |
| ---------- | ------------------ |
| 居民身份证 | 含临时居民身份证   |
| 护照       | 中华人民共和国护照 |
| 军官证     | 军官证、文职干部证 |
| 义务兵证   | —                  |
| 士官证     | —                  |
| 文职人员证 | —                  |
| 职工证     | —                  |
| 武警警官证 | 含武警士兵证       |
| 海员证     | —                  |

### 港澳台居民

| 证件类型               | 说明   |
| ---------------------- | ------ |
| 港澳居民来往内地通行证 | 回乡证 |
| 台湾居民来往大陆通行证 | 台胞证 |
| 港澳台居民居住证       | —      |

### 外籍旅客

| 证件类型         | 说明                   |
| ---------------- | ---------------------- |
| 护照             | —                      |
| 驻华外交人员证   | 外交部签发             |
| 外国人永久居留证 | 含外国人永久居留身份证 |

### 十六周岁以下未成年人

除上述证件外，还可使用：

- 出生医学证明
- 户口簿
- 学生证
- 户口所在地公安机关出具的身份证明

## 二、临时乘机证明

如身份证件丢失或遗忘，可通过以下方式办理临时乘机证明：

### 电子临时乘机证明

通过微信/支付宝搜索"民航临时乘机证明"小程序，在线申请，生成电子二维码，在机场安检通道扫码验证。

- 有效期：15天
- 适用：国内航班
- 免费办理

### 机场公安办理

前往机场公安派出所或临时身份证明办理柜台，提供个人信息，现场打印纸质临时乘机证明。

---

## 三、证件注意事项

### 一致性要求

购票、值机、安检必须使用**同一个有效乘机身份证件**，信息不一致将无法乘机。

### 证件过期

居民身份证过期不能继续使用，需提前办理临时乘机证明或使用其他有效证件。

### 国内使用护照

中华人民共和国护照可以作为有效乘机身份证件，办理国内航班购票、值机、安检手续。

---

## 四、国际航班证件要求

| 证件     | 要求                                                |
| -------- | --------------------------------------------------- |
| 护照     | 有效期需在行程最后一个航班出发日期之后至少6个月以上 |
| 签证     | 根据目的地国家要求提前办理                          |
| 转机签证 | 部分国家转机也需签证，请提前确认                    |

### 行前检查清单

- 确保护照有效期充足
- 确认目的地国家签证要求（含转机）
- 检查目的地国家入境限制及健康要求
- 儿童/婴儿也需持有本人护照或有效证件
- 港澳台居民前往国外需持有效护照及目的地签证
